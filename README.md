# progmatoz-template
This is a template project for Java microservices. 
<br/>
Among the technologies used, the main ones are:
<br/>
- Java 16 (OpenJDK)
- Spring Boot 2.4.4
- Lombok
- Swagger 3.0.0
- Spotbugs

Main Features:
<br/>
- CRUD mock template requests
- Log all requests and responses data
- Log in JSON format
- Hateoas
